# kanzaiya

## サイト概要
### サイトテーマ
会社側の在庫管理と顧客側が在庫確認と注文できるサイト
オリジナル題材


### テーマを選んだ理由
現在の勤めている会社が、注文を頂いた際に、在庫があるかわざわざ倉庫まで行き確認しなければならない状態で
手間な為、パソコンで簡単に調べられる機能が欲しいと思ったからです。
それとプラスで、お客さんがWEBで注文が出来たらお客様も手軽で、楽なのではないかと思ったのでこのテーマに決めました。

### ターゲットユーザ
建設関係のお客様、一般の方

### 主な利用シーン
今すぐ物が欲しいが、在庫があるのかどうか確認したい等
